Mac下好用的命令行工具(2026版)

发表于 2026-02-09 分类于 数字化工具 Waline： 本文字数： 1.4k 阅读时长 ≈ 2 分钟

结合对 Synology NAS、Vim/fzf 以及 知识管理 (PKM) 的兴趣，以下是 2026 年 macOS 平台上最值得推荐的命令行工具，按功能类别进行精选：

一、 核心基础与文件导航

这些工具是高效终端环境的基石，特别适合习惯使用 fzf 的用户。

  • Homebrew (brew)：必备的软件包管理器。
  • eza (替代 ls)：现代化的文件列表工具。支持图标显示、Git 状态集成以及更清晰的颜色分级。
    • 安装：brew install eza
  • fd (替代 find)：比原生 find 快且语法更简单，默认忽略 .gitignore 中的文件。
    • 安装：brew install fd
  • ripgrep (rg) (替代 grep)：极速的文本搜索工具，是目前性能最强的文件内容搜索工具。
    • 安装：brew install ripgrep
  • bat (替代 cat)：带有语法高亮、代码折行和 Git 修改标记的查看器。
    • 安装：brew install bat

二、 生产力与数据处理

针对您在 制造工程文档处理 和 知识管理 方面的需求：

  • Pandoc：文档转换的“瑞士军刀”。支持 Markdown、Word、PDF、LaTeX 等数十种格式互转，非常适合处理工艺流程说明书。
    • 安装：brew install pandoc
  • jq / yq：命令行 JSON 和 YAML 处理器。在调试 Synology API 或处理结构化文档时极度高效。
    • 安装：brew install jq yq
  • tldr：简化版的帮助手册（man pages）。快速获取命令的核心用法示例，而非冗长的文档。
    • 安装：brew install tldr
  • Zoxide (替代 cd)：智能路径跳转工具。它会记录您访问过的目录，只需输入 z 部分目录名 即可瞬间跳转。
    • 安装：brew install zoxide

三、 系统监控与磁盘管理

  • btop / htop：美观且功能强大的实时系统监视器，支持查看 CPU、内存、网络及进程。
    • 安装：brew install btop
  • ncdu：磁盘占用分析器。以交互式菜单形式显示哪些文件占用了空间，适合清理 Mac 和 Synology 挂载盘。
    • 安装：brew install ncdu
  • Mole：2026 年流行的一款针对 macOS 的“深层清理”命令行工具，用于清除系统冗余缓存。

四、 网络、存储与 NAS 同步

考虑到您对 Synology NAS 的深度使用：

  • rclone：云存储同步管理的“终极工具”。支持将 Synology、Google Drive、S3 等挂载为本地磁盘，或进行加密备份。
    • 安装：brew install rclone
  • yt-dlp：目前最全能的视频下载工具，支持 YouTube 及国内主流视频平台。
    • 安装：brew install yt-dlp
  • Ollama：在本地终端运行大型语言模型（如 Llama 3, Mistral）。
    • 安装：brew install ollama

五、 终端增强 (2026 趋势)

  • Warp：基于 Rust 的现代化终端模拟器。自带 AI 命令预测和块状输出（Blocks），极大提升了传统 CLI 的交互体验。
  • Starship：极其迅速、可高度定制的跨平台 Shell 提示符。能根据您当前的目录（如进入 Git 仓库、Python 环境）自动切换显示状态。
    • 安装：brew install starship

推荐组合建议表

传统命令 推荐替代方案 核心优势
ls eza 色彩、图标、Git 集成
find fd 速度快、语法直观、默认过滤
grep rg 性能极强、支持多核搜索
cat bat 语法高亮、代码预览
cd zoxide 历史路径模糊跳转
